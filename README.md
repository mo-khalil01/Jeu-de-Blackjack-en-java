# Jeu de Blackjack en Java

Projet réalisé par **Florentin Briand**, **Corentin Quatrevaux** et **Khalil Mohammad** dans le cadre du cours **Méthodes de conception** de troisième année de **Licence Informatique** à l’Université de Caen.

Ce projet consiste en le développement d’un jeu de **Blackjack interactif en Java**, structuré selon le modèle **MVC (Modèle–Vue–Contrôleur)** afin de séparer la logique, la vue graphique et le contrôle.

## Compilation et exécution

### Création de la librairie de cartes
Dans le répertoire `card` :
```bash
ant dist
``` 
Cette commande permet de créer la librairie utilisée par l’application.

### Exécution du jeu
* Version graphique
```bash
ant runS
```
* Version terminal
```bash
ant runT
```
